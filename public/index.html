Distribution Tracker

API Key Required

Distribution Tracker uses Polygon.io for live dividend & price data. A free API key is required (no credit card).

  1. Sign up for free at polygon.io
  2. Copy your API key from the dashboard
  3. In the app folder, create a file called .env containing:
    POLYGON_KEY=your_api_key_here
  4. Restart the server: npm start
